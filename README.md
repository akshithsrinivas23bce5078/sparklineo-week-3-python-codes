# sparklineo-week-3-python-codes
Collection of Python Programs using Basic Data Types like String, list, tuple, dictionary:
  1. Reverse a string: This program takes a string input and prints its reverse.
  2. Find maximum number in a list: This program takes a list of numbers and displays the maximum value.
  3. Student data using dictionary: This program stores and manages student details using a dictionary with options like add, search, update, delete, and display
